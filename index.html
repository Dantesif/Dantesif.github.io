Command to edit the Static IP config file:

sudo nano /etc/dhcpcd.conf

Paste this in to the file and edit where needed:-

interface eth0

static ip_address=192.168.0.10/24

static routers=192.168.0.1

static domain_name_servers=192.168.0.1

interface eth1

static ip_address=192.168.0.10/24

static routers=192.168.0.1

static domain_name_servers=192.168.0.1

Ctrl+X S Enter

Commands to update the Pi:-

sudo apt-get update

sudo apt-get upgrade

Command to install Pi-Hole:-

sudo curl -sSL http://install.pi-hole.net | bash
